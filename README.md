# Campus Network System

A simulated campus network infrastructure developed using Cisco Packet Tracer.

## Overview

This project was created to demonstrate the design and implementation of a campus-wide network system for an educational institution.

The network simulation includes multiple interconnected departments, routers, switches, and end devices to ensure efficient communication across the campus environment.

It serves as a beginner-to-intermediate level networking project for learning network design, configuration, and simulation concepts.

## Features

* Structured campus network topology
* Multiple department connectivity
* Router and switch configuration
* End-to-end device communication
* Scalable network design
* Cisco Packet Tracer simulation environment

## Technologies Used

* Cisco Packet Tracer
* Computer Networking Concepts
* Routing and Switching
* IP Addressing

## How to Run the Project

1. Install Cisco Packet Tracer on your computer.

2. Clone the repository:

```bash
git clone https://github.com/your-username/Campus-Network-System.git
```

3. Open the `.pkt` file using Cisco Packet Tracer.

4. Run the simulation and test network connectivity.
