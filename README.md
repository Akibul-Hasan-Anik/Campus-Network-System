# Campus Network System

A simulated campus-wide network infrastructure developed using Cisco Packet Tracer.

## Overview

This project was created as a beginner-friendly demonstration of how a campus network system is designed and implemented using Cisco Packet Tracer. It contains a complete network topology connecting multiple departments, routers, switches, and end devices. It serves as an introduction to computer networking, network simulation, routing and switching concepts, and campus network architecture.

## Features

* Structured campus network topology
* Multiple department connectivity
* Router and switch configuration
* End-to-end device communication

## Technologies Used

* Cisco Packet Tracer
* Computer Networking
* Routing and Switching
* IP Addressing
